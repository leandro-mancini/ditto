# Instalação

## Introdução

O Ditto é uma biblioteca flexível que fornece suporte para **React**, **React Native** e **Web Components** utilizando **Preact**. Com o Ditto, você pode criar interfaces de usuário consistentes e escaláveis, seja em aplicações web, mobile ou como componentes reutilizáveis em qualquer lugar do ecossistema JavaScript.

## Pré-requisitos

Antes de começar a instalar o Ditto, verifique se você possui as seguintes dependências instaladas em seu ambiente de desenvolvimento:

- Node.js versão 14.x ou superior
- NPM ou Yarn para gerenciamento de pacotes
- Um projeto configurado com React, React Native ou Preact (dependendo do seu caso de uso)

## Instalando o Ditto

### Para Aplicações React

Para utilizar o Ditto em um projeto React, basta instalar o pacote via NPM ou Yarn:

```bash
npm install @dittox/styled-system
```

Ou, se você preferir Yarn:

```bash
yarn add @dittox/styled-system
```
