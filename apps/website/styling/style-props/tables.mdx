## Border Spacing

Controle a propriedade `border-spacing` de uma tabela.

```jsx
<ditto.table borderSpacing="2">
  <tbody>
    <tr>
      <td>Cell 1</td>
      <td>Cell 2</td>
    </tr>
  </tbody>
</ditto.table>
```

| Prop            | CSS Property     | Token Category |
| --------------- | ---------------- | -------------- |
| `borderSpacing` | `border-spacing` | `spacing`      |

## Caption Side

Use a propriedade captionSide para definir o lado da legenda de uma tabela.

```jsx
<table>
  <ditto.caption captionSide="bottom">This is a caption</ditto.caption>
  <tbody>
    <tr>
      <td>Cell 1</td>
      <td>Cell 2</td>
    </tr>
  </tbody>
</table>
```

| Prop          | CSS Property   | Token Category |
| ------------- | -------------- | -------------- |
| `captionSide` | `caption-side` | -              |
