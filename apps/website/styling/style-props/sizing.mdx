## Width

Use a propriedade `width` ou `w` para definir a largura de um elemento.

```jsx
// hardcoded values
<Box width="64px" />
<Box w="4rem" />

// token values
<Box width="5" />
<Box w="5" />
```

### Fractional width

Você pode definir larguras fracionadas usando a propriedade width ou w.

Os valores podem estar nos seguintes intervalos:

- Terços: 1/3 até 2/3
- Quartos: 1/4 até 3/4
- Quintos: 1/5 até 4/5
- Sextos: 1/6 até 5/6
- Duzavos: 1/12 até 11/12

```jsx
// half width
<Flex>
  <Box width="1/2" />
  <Box width="1/2" />
</Flex>

// thirds
<Flex>
  <Box width="1/3" />
  <Box width="2/3" />
</Flex>

// fourths
<Flex>
  <Box width="1/4" />
  <Box width="3/4" />
</Flex>

// fifths
<Flex>
  <Box width="1/5" />
  <Box width="4/5" />
</Flex>

// sixths
<Flex>
  <Box width="1/6" />
  <Box width="5/6" />
</Flex>

// twelfths
<Flex>
  <Box width="3/12" />
  <Box width="9/12" />
</Flex>
```

### Viewport width

Use os valores modernos de largura de janela de visualização dvw, svw, lvw.

> dvw mapeia para 100dvw, svw mapeia para 100svw, lvw mapeia para 100lvw.

```jsx
<Box width="dvw" />
<Box w="dvw" /> // shorthand
```

| Prop         | CSS Property | Token Category |
| ------------ | ------------ | -------------- |
| `w`, `width` | `width`      | `sizes`        |
