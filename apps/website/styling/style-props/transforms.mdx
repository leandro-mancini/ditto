## Scale

Use a propriedade `scale` para controlar a escala de um elemento.

```jsx
<Box scale="1.2" />
```

Quando a propriedade scale é definida como auto, as propriedades scaleX e scaleY são usadas para controlar a escala do elemento.

| Prop    | CSS Property | Token Category |
| ------- | ------------ | -------------- |
| `scale` | `scale`      | -              |

## Scale X

Use a propriedade `scaleX` para controlar a propriedade scaleX de um elemento. Isso requer que a propriedade `scale` seja definida como `auto`.

```jsx
<Box scale="auto" scaleX="1.3" />
```

| Prop     | CSS Property | Token Category |
| -------- | ------------ | -------------- |
| `scaleX` | `--scale-x`  | -              |

## Scale Y

Use a propriedade `scaleY` para controlar a propriedade scaleY de um elemento. Isso requer que a propriedade `scale` seja definida como `auto`.

```jsx
<Box scale="auto" scaleY="0.4" />
```

| Prop     | CSS Property | Token Category |
| -------- | ------------ | -------------- |
| `scaleY` | `--scale-y`  | -              |

## Translate

Use a propriedade `translate` para controlar a propriedade de translação de um elemento.

```jsx
<Box translate="40px" />
<Box translate="50% -40%" />
```

Quando a propriedade `translate` é definida como `auto`, as propriedades `translateX` e `translateY` são usadas para controlar a translação do elemento.

| Prop        | CSS Property | Token Category |
| ----------- | ------------ | -------------- |
| `translate` | `translate`  | -              |

## Translate X

Use a propriedade `translateX` para controlar a propriedade translateX de um elemento. Isso requer que a propriedade `translate` seja definida como `auto`.

```jsx
// valores fixos
<Box translate="auto" translateX="50%" />
<Box translate="auto" translateX="20px" />

// valores de token
<Box translate="auto" translateX="4" />
<Box translate="auto" translateX="-10" />
```

| Prop         | CSS Property    | Token Category |
| ------------ | --------------- | -------------- |
| `translateX` | `--translate-x` | `spacing`      |

## Translate Y

Use a propriedade `translateY` para controlar a propriedade translateY de um elemento. Isso requer que a propriedade `translate` seja definida como `auto`.

```jsx
// valores fixos
<Box translate="auto" translateY="-40%" />
<Box translate="auto" translateY="4rem" />

// valores de token
<Box translate="auto" translateY="4" />
<Box translate="auto" translateY="-10" />
```

| Prop         | CSS Property    | Token Category |
| ------------ | --------------- | -------------- |
| `translateY` | `--translate-y` | `spacing`      |
