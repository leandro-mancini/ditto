## Accent Color

Use a propriedade `accentColor` para definir a cor de acento para controles de interface do usuário gerados pelo navegador.

```jsx
// valor fixo
<label>
  <ditto.input type="checkbox" accentColor="#3b82f6" />
</label>

// valor de token
<label>
  <ditto.input type="checkbox" accentColor="blue.500" />
</label>
```

| Prop          | CSS Property   | Token Category |
| ------------- | -------------- | -------------- |
| `accentColor` | `accent-color` | `colors`       |

## Appearance

Use a propriedade `appearance` para definir a aparência de um elemento.

```jsx
<ditto.select appearance="none">
  <option value="1">Option 1</option>
  <option value="2">Option 2</option>
  <option value="3">Option 3</option>
</ditto.select>
```

| Prop         | CSS Property | Token Category |
| ------------ | ------------ | -------------- |
| `appearance` | `appearance` | -              |

## Scrollbar

## Scroll Snap Type

Use a propriedade `scrollSnapType` para controlar como os pontos de encaixe são aplicados em um contêiner de rolagem.

```jsx
<Box maxH="100px" overflowY="auto" scrollSnapType="x">
  Scroll container with x snap type
</Box>
```

| Value  |                                      |
| ------ | ------------------------------------ |
| `none` | `none`                               |
| `x`    | `x var(--scroll-snap-strictness)`    |
| `y`    | `y var(--scroll-snap-strictness)`    |
| `both` | `both var(--scroll-snap-strictness)` |

## Scroll Snap Strictness

Use a propriedade `scrollSnapStrictness` para definir o rigor do encaixe de rolagem de um elemento. Isso requer que `scrollSnapType` esteja definido como `x`, `y` ou `both`.

Os valores podem ser `mandatory` ou `proximity` e mapeiam para `var(--scroll-snap-strictness)`.

```jsx
<Box maxH="100px" overflowY="auto" scrollSnapStrictness="proximity">
  <Box>Item 1</Box>
  <Box>Item 2</Box>
</Box>
```

| Prop                   | CSS Property               | Token Category |
| ---------------------- | -------------------------- | -------------- |
| `scrollSnapStrictness` | `--scroll-snap-strictness` | -              |
