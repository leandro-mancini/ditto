## Uso

Por padrão, `Stack` aplica `flex-direction: column` e `gap: 8px` aos seus filhos.

```jsx
import { HStack, Stack, VStack } from '@dittox/react';

<Stack>
  <div />
  <div />
</Stack>;
```

## Exemplos

### Horizontal

Use a propriedade `direction` para alterar a direção da pilha.

```
<ExampleTabs name="stack-horizontal" />
```

```tsx
export const StackHorizontal = () => {
  return (
    <Stack direction="row" h="20">
      <Box />
      <Box />
      <Box />
    </Stack>
  );
};
```

## `HStack`

Alternativamente, você pode usar o `HStack` para criar uma pilha horizontal e alinhar seus filhos horizontalmente.

```
<ExampleTabs name="stack-with-hstack" />
```

```tsx
export const StackWithHstack = () => {
  return (
    <HStack>
      <Box h="10" />
      <Box h="5" />
      <Box h="20" />
    </HStack>
  );
};
```

## `VStack`

Use o `VStack` para criar uma pilha vertical e alinhar seus filhos verticalmente.

```
<ExampleTabs name="stack-with-vstack" />
```

```tsx
export const StackWithVstack = () => {
  return (
    <VStack>
      <Box w="50%" h="20" />
      <Box w="25%" h="20" />
      <Box w="100%" h="20" />
    </VStack>
  );
};
```

## Separador

Use a propriedade `separator` para adicionar um separador entre os itens da pilha.

```
<ExampleTabs name="stack-with-separator" />
```

```tsx
export const StackWithSeparator = () => {
  return (
    <Stack separator={<StackSeparator />}>
      <Box h="20" />
      <Box h="20" />
      <Box h="20" />
    </Stack>
  );
};
```

## Direção Responsiva

Use a propriedade `direction` para alterar a direção da pilha de forma responsiva.

```
<ExampleTabs name="stack-with-responsive-direction" />
```

```tsx
export const StackWithResponsiveDirection = () => {
  return (
    <Stack direction={{ base: 'column', md: 'row' }} gap="10">
      <Box boxSize="20" />
      <Box boxSize="20" />
      <Box boxSize="20" />
    </Stack>
  );
};
```
