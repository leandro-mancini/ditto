## Uso

```jsx
import { AbsoluteCenter, Center, Circle, Square } from '@dittox/react';

<Center bg="tomato" h="100px" color="white">
  Center
</Center>;
```

## Exemplos

### Ícone

O componente `Center` pode ser usado para criar molduras ao redor de ícones ou números.

```
<ExampleTabs name="center-with-icons" />
```

```tsx
export const CenterWithIcons = () => {
  return (
    <HStack>
      <Center w="40px" h="40px" bg="tomato" color="white">
        <LuPhone />
      </Center>

      <Center w="40px" h="40px" bg="tomato" color="white">
        <Box as="span" fontWeight="bold" fontSize="lg">
          1
        </Box>
      </Center>
    </HStack>
  );
};
```

### Center com Inline

Use `inline` para mudar o display para `inline-flex`.

```
<ExampleTabs name="center-with-inline" />
```

```tsx
export const CenterWithInline = () => {
  return (
    <Link href="#">
      <Center inline gap="4">
        <Box>Ditto Design System</Box>
        <LuArrowRight />
      </Center>
    </Link>
  );
};
```

### Square

Centraliza seu filho com base na propriedade `size` (largura e altura).

```
<ExampleTabs name="center-with-square" />
```

```tsx
export const CenterWithSquare = () => {
  return (
    <Square size="10" bg="purple.700" color="white">
      <LuPhoneForwarded />
    </Square>
  );
};
```

### Circle

Centraliza seu filho com base na propriedade `size` e cria um círculo ao redor dele.

```
<ExampleTabs name="center-with-circle" />
```

```tsx
export const CenterWithCircle = () => {
  return (
    <Circle size="10" bg="blue.700" color="white">
      <LuPhoneForwarded />
    </Circle>
  );
};
```

### AbsoluteCenter

`AbsoluteCenter` centraliza em relação ao seu pai usando a estratégia `position: absolute`. Passe a propriedade `axis` para mudar o eixo de alinhamento.

```
<ExampleTabs name="center-with-absolute" />
```

```tsx
export const CenterWithAbsolute = () => {
  return (
    <Box position="relative" h="100px">
      <AbsoluteCenter bg="tomato" p="4" color="white" axis="both">
        <LuPhone />
      </AbsoluteCenter>
    </Box>
  );
};
```
