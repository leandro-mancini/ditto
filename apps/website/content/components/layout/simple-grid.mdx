## Uso

O componente `SimpleGrid` permite que você crie layouts de grade responsivos
com facilidade.

```jsx
import { SimpleGrid } from '@dittox/react';

<SimpleGrid>
  <Box />
  <Box />
</SimpleGrid>;
```

## Exemplos

### Colunas

Especifique o número de colunas para o layout de grade usando a propriedade `columns`.


### Auto-responsivo

Torne a grade responsiva e ajustável automaticamente sem precisar passar colunas, usando a propriedade `minChildWidth`. Isso usa internamente o css grid `auto-fit` e `minmax()`.
