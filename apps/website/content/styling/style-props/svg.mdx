title: SVG

---

## Fill

Use a propriedade `fill` para definir a cor de preenchimento de um elemento SVG.

```jsx
<ditto.svg fill="blue.500">
  <path d="..." />
</ditto.svg>
```

| Prop   | CSS Property | Token Category |
| ------ | ------------ | -------------- |
| `fill` | `fill`       | `colors`       |

## Stroke

Use a propriedade `stroke` para definir a cor do traço de um elemento SVG.

```jsx
<ditto.svg stroke="blue.500">
  <path d="..." />
</ditto.svg>
```

| Prop     | CSS Property | Token Category |
| -------- | ------------ | -------------- |
| `stroke` | `stroke`     | `colors`       |

## Stroke Width

Use a propriedade `strokeWidth` para definir a largura do traço de um elemento SVG.

```jsx
<ditto.svg strokeWidth="1px">
  <path d="..." />
</ditto.svg>
```

| Prop          | CSS Property   | Token Category |
| ------------- | -------------- | -------------- |
| `strokeWidth` | `stroke-width` | `borderWidths` |
