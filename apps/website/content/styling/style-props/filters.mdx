## Filter

Use a propriedade `filter` para aplicar efeitos visuais como desfoque ou mudança de cor a um elemento.

```jsx
<Box filter="blur(5px)" />
<Box filter="grayscale(80%)" />
```

| Prop     | CSS Property | Token Category |
| -------- | ------------ | -------------- |
| `filter` | `filter`     | -              |

## Blur

Use a propriedade `blur` para aplicar um efeito de desfoque a um elemento. O requisito para essa propriedade é definir a propriedade `filter` como `auto`.

```jsx
// valor fixo
<Box filter="auto" blur="5px" />

// valor de token
<Box filter="auto" blur="sm" />
```

| Prop   | CSS Property | Token Category |
| ------ | ------------ | -------------- |
| `blur` | `--blur`     | `blurs`        |

## Backdrop Filter

```jsx

<Box backdropFilter="grayscale(80%)" />
```

| ---------------- | ----------------- | -------------- |
| `backdropFilter` | `backdrop-filter` | -              |
