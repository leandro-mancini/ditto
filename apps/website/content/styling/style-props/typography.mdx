## Font Family

Use a propriedade `fontFamily` para definir a família de fontes de um elemento de texto.

```jsx
<Text fontFamily="mono">Olá Mundo</Text>
```

| Prop         | CSS Property  | Token Category |
| ------------ | ------------- | -------------- |
| `fontFamily` | `font-family` | `fonts`        |

## Font Size

Use a propriedade `fontSize` para definir o tamanho da fonte de um elemento de texto.

```jsx
// valores fixo
<Text fontSize="12px">Ditto</Text>
<Text fontSize="10rem">Ditto</Text>

// valores de token
<Text fontSize="xs">Ditto</Text>
<Text fontSize="4xl">Ditto</Text>
<Text fontSize="5xl">Ditto</Text>
```

| Prop       | CSS Property | Token Category |
| ---------- | ------------ | -------------- |
| `fontSize` | `font-size`  | `fonts`        |

## Text Styles

```jsx
<Text textStyle="lg">Ditto</Text>
<Text textStyle="xl">Ditto</Text>
<Text textStyle="2xl">Ditto</Text>
<Text textStyle="3xl">Ditto</Text>

<Text textStyle="5xl">Ditto</Text>
```

| Prop        | Config             |
| ----------- | ------------------ |
| `textStyle` | `theme.textStyles` |

## Line Clamp

Use a propriedade `lineClamp` para truncar texto em múltiplas linhas. Defina `lineClamp` como `none` para desativar o truncamento.

```jsx
<Text lineClamp="2">Lorem ipsum dolor sit amet...</Text>

// reverter truncamento
<Text lineClamp="none">Lorem ipsum dolor sit amet...</Text>
```

| Prop        | CSS Property        | Token Category |
| ----------- | ------------------- | -------------- |
| `lineClamp` | `webkit-line-clamp` | none           |
