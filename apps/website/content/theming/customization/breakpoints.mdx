## Exemplo

Aqui está um exemplo de como personalizar os pontos de interrupção no Ditto.

```tsx title="theme.ts"
import { createSystem, defaultConfig, defineConfig } from '@ditto/react';

const config = defineConfig({
  theme: {
    breakpoints: {
      tablet: '992px',
      desktop: '1200px',
      wide: '1400px',
    },
  },
});

export default createSystem(defaultConfig, config);
```

## Uso

Ao usar propriedades responsivas, faça referência aos novos pontos de interrupção.

```tsx
<Box fontSize={{ base: '16px', tablet: '18px', desktop: '20px' }}>Olá</Box>
```
