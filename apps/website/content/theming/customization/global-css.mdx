## Exemplo

Aqui está um exemplo de como personalizar o CSS global no Ditto.

```tsx title="theme.ts"
import { createSystem, defaultConfig, defineConfig } from '@dittox/react';

const customConfig = defineConfig({
  globalCss: {
    '*::placeholder': {
      opacity: 1,
      color: 'fg.subtle',
    },
    '*::selection': {
      bg: 'green.200',
    },
  },
});

export const system = createSystem(defaultConfig, customConfig);
```
