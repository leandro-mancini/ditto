## Exemplo

Aqui está um exemplo de como personalizar tamanhos no Ditto.

```tsx title="theme.ts"
import { createSystem, defaultConfig, defineConfig } from '@dittox/react';

const customConfig = defineConfig({
  theme: {
    tokens: {
      sizes: {
        '1/7': { value: '14.285%' },
        '2/7': { value: '28.571%' },
        '3/7': { value: '42.857%' },
      },
    },
  },
});

export const system = createSystem(defaultConfig, customConfig);
```

## Uso

Defina o valor de qualquer propriedade relacionada a tamanho, como `width`, `height`, `minWidth`, `minHeight`, `maxWidth`, `maxHeight`, etc., para o token sizes.

```tsx
<Box width="1/7" height="2/7" />
```
