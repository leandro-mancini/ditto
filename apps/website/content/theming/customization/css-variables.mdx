## Raiz das Variáveis

Aqui está um exemplo de como personalizar o seletor ao qual as variáveis CSS de tokens
são aplicadas.

```tsx title="theme.ts"
import { createSystem, defaultConfig, defineConfig } from '@dittox/react';

const customConfig = defineConfig({
  cssVarsRoot: ':where(:root, :host)',
});

export const system = createSystem(defaultConfig, customConfig);
```

As variáveis CSS emitidas agora serão aplicadas ao elemento `root` e `host`.

```css
:where(:root, :host) {
  --ditto-colors-gray-100: #e6f2ff;
  --ditto-colors-gray-200: #bfdeff;
  --ditto-colors-gray-300: #99caff;
}
```

## Prefixo das Variáveis

Aqui está um exemplo de como personalizar o prefixo das variáveis CSS emitidas.

```tsx
import { createSystem, defaultConfig, defineConfig } from '@dittox/react';

const customConfig = defineConfig({
  cssVarsPrefix: 'mancini',
});

export const system = createSystem(defaultConfig, customConfig);
```

As variáveis CSS emitidas agora usarão o prefixo `mancini`.

```css
:where(:root, :host) {
  --mancini-colors-gray-100: #e6f2ff;
  --mancini-colors-gray-200: #bfdeff;
  --mancini-colors-gray-300: #99caff;
}
```
