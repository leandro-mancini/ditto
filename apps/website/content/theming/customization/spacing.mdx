## Exemplo

Aqui está um exemplo de como personalizar o espaçamento no Ditto.

```tsx title="theme.ts"
import { createSystem, defaultConfig, defineConfig } from '@dittox/react';

const customConfig = defineConfig({
  theme: {
    tokens: {
      spacing: {
        '128': { value: '32rem' },
        '144': { value: '36rem' },
      },
    },
  },
});

export const system = createSystem(defaultConfig, customConfig);
```

## Uso

Aqui está como usar o espaçamento personalizado no Ditto.

```tsx
<Box margin="128" />
```
