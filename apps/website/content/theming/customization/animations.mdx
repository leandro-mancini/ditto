## Exemplo

Aqui está um exemplo de como personalizar keyframes no Ditto.

```tsx title="theme.ts"
import { createSystem, defaultConfig, defineConfig } from '@dittox/react';

const config = defineConfig({
  theme: {
    keyframes: {
      wiggle: {
        '0%, 100%': { transform: 'rotate(-3deg)' },
        '50%': { transform: 'rotate(3deg)' },
      },
    },
    tokens: {
      animations: {
        wiggle: { value: 'wiggle 1s infinite' },
      },
    },
  },
});

export const system = createSystem(defaultConfig, config);
```

## Uso

Defina o valor da propriedade de estilo `animationName` ou `animation` para aplicar os keyframes e animação, respectivamente.

```tsx
<Box animation="wiggle" />
<Box animationName="wiggle" animationDuration="fast" animationIterationCount="infinite" />
```
